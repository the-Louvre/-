Amazon Reviews 数据挖掘（课程作业提交版）

对亚马逊商品评论进行清洗 → 探索性分析（EDA）→ 情感/评分建模 → 主题发现 → 可视化的完整流程。
目标是给出可复现的实验与清晰的结论，便于助教复查与打分。

⸻

   项目简介
	•	数据来源：Amazon Reviews（含 overall 评分、reviewText 正文、summary 摘要、asin 商品ID、reviewerID 用户ID、helpful 有用票等字段）。
	•	研究问题
	1.	评论文本的情感倾向能否被准确判别？（二分类/多分类）
	2.	文本能否预测评分（overall）？（回归）
	3.	评论中潜在的**主题（Topic）**是什么？（LDA/BERTopic）
	4.	（可选）有用票能否被初步预测？（回归/二分类）
	•	主要贡献
	•	标准化清洗与可复现管线（脚本/Notebook）
	•	多模型对比（TF-IDF + 传统模型 vs. 预训练文本向量）
	•	完整的评估指标与可视化图表（混淆矩阵、PR曲线、主题词云等）
